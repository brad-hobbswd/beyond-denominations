## Overview

**Beyond Denominations: Returning to the Apostles' Doctrine** is a Scripture-centered journey through the Book of Acts, inviting readers to look beyond traditions, movements, and denominational identities to rediscover the pattern established by Jesus Christ and faithfully proclaimed by His apostles.

Rather than comparing churches or debating history, this project follows the apostolic witness from Jerusalem throughout the Roman world, allowing Scripture to speak with clarity and consistency.

The goal is simple:

> Return to the beginning.

> Return to the apostles.

> Return to Jesus Christ.

## Purpose

This project is not an argument against sincere believers.

It is an invitation to return to the witness preserved in the Book of Acts and rediscover the simplicity, unity, and beauty of the apostolic church.

The central question remains the same one asked in Jerusalem:

> "Men and brethren, what shall we do?"
